# prompt-engineering-portfolio
A hands-on collection of Before vs After prompt transformations — showcasing prompt engineering techniques learned during Google Cloud Prompt Engineering certification.

## The Golden Formula

Every great prompt follows this structure:

Role → Who should the AI act as?
Task → What do you want it to do?
Context → What's the background info?
Format → How should the answer look?
Constraints → Any limits? (word count, tone, etc.)

### Example
❌ `Write a bio`

✅ `You are a LinkedIn coach [Role].
Write a bio for a CS student [Task].
He has a Google Cloud Badge and wants AI internships [Context].
Keep it 3 lines [Format].
Make it confident, no buzzwords [Constraints].`
