# Zero-Shot Prompting — LinkedIn Bio

##  Bad Prompt
Write my LinkedIn bio

##  AI Output (Bad)
Turning curiosity into skills and skills into opportunities. 
Passionate about technology, innovation, and lifelong learning. 
Currently focused on growing in Software Development, AI, and 
Cloud Computing while building a strong foundation for a successful tech career.


---

## Good Prompt
You are a LinkedIn profile coach. Write a 3-line bio 
for a computer science student from India who has a 
Google Cloud Prompt Engineering badge and is looking 
for AI/ML internships. Make it confident and professional.

## AI Output (Good)
Aspiring AI/ML Engineer | Computer Science Student | Google Cloud Prompt Engineering Badge Holder
Building expertise in Python, Machine Learning, SQL, and Cloud Computing through projects and continuous learning.
Seeking AI/ML internship opportunities to develop intelligent solutions and gain industry experience.


---

## 🧠 What Changed
- Added role → LinkedIn profile coach
- Added context → CS student, Google Cloud badge
- Added constraints → 3 lines, confident tone
