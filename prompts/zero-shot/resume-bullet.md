# Few-Shot Prompting — Resume Bullet Points

## Bad Prompt
Improve my resume

## AI Output (Bad)
- Worked on machine learning projects.
- Helped with prompt engineering tasks.
- Learned cloud technologies.
- Participated in software development activities.

---

## Good Prompt
Improve these resume bullet points to sound more
impactful using action verbs and numbers.

Examples:
"worked on a project" → "Built and deployed a web app serving 200+ users"
"helped in cloud tasks" → "Optimized Google Cloud pipelines reducing cost by 30%"

Now improve these:
- "did machine learning stuff"
- "worked on prompt engineering"

## AI Output (Good)
- Developed and evaluated machine learning models, improving prediction accuracy by up to 15% on benchmark datasets.
- Designed and tested 50+ AI prompts for content generation and task automation, increasing response quality and consistency.
- Applied data preprocessing and feature engineering techniques to improve model performance.
- Collaborated with AI tools to create structured prompts for real-world use cases and productivity workflows.

---

##  What Changed
- Used Few-Shot Prompting by providing examples before the task.
- Examples taught the AI the desired writing style and level of detail.
- AI generated stronger action verbs, measurable achievements, and professional resume language.
- Output became more specific, impactful, and recruiter-friendly.
