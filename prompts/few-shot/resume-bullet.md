# Few-Shot Prompting — Resume Bullet Points

## Bad Prompt

Improve my resume

## AI Output (Bad)

* Worked on machine learning projects.
* Helped with prompt engineering tasks.
* Participated in AI-related activities.
* Learned cloud technologies and programming skills.

---

##  Good Prompt

Improve these resume bullet points to sound more impactful using action verbs and numbers.

Examples:
"worked on a project" → "Built and deployed a web app serving 200+ users"

"helped in cloud tasks" → "Optimized Google Cloud pipelines reducing cost by 30%"

Now improve these:

* "did machine learning stuff"
* "worked on prompt engineering"

## AI Output (Good)

* Developed and evaluated machine learning models, improving prediction accuracy by 15% on sample datasets.
* Designed and tested 50+ AI prompts, increasing response quality and consistency across tasks.
* Applied data preprocessing techniques to enhance model performance and reliability.
* Collaborated with AI tools to automate workflows, reducing manual effort by 25%.

---

##  What Changed

* Used Few-Shot Prompting by providing examples before the task.
* Examples taught the AI the expected writing style and structure.
* AI followed the demonstrated pattern automatically.
* Output became more specific, measurable, and professional.
* Action verbs and quantified achievements made the resume bullets stronger.
