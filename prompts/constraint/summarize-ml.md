# Constraint Prompting — Summarize Machine Learning

## ❌ Bad Prompt
Summarize machine learning

## 💬 AI Output (Bad)

Machine learning is a branch of artificial intelligence that enables computers to learn from data and improve their performance without being explicitly programmed.
It is used in applications such as recommendation systems, image recognition, natural language processing, and predictive analytics.

---

## ✅ Good Prompt
Summarize Machine Learning in exactly 5 bullet points.

Rules:
- Each bullet must be under 15 words
- No technical jargon
- Start each bullet with an action verb
- Target audience: a school student
- Do NOT mention math or algorithms

## 💬 AI Output (Good)
-> Learn from examples instead of following fixed instructions.
-> Recognize patterns in everyday information.
-> Predict future outcomes using past experiences.
-> Improve performance through repeated practice.
-> Solve real-world problems more efficiently.

---

## 🧠 What Changed
- Added exact number → 5 bullet points only
- Added word limit → under 15 words each
- Added negative constraint → no jargon, no math
- Added audience → school student
- Output was clean, simple and actually useful
